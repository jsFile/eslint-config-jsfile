# eslint-config-jsfile
ESLint shareable config for the JsFile style guide

## Install

```
$ npm install --save-dev eslint eslint-config-jsfile
```


## Usage

Add some ESLint config to your `.eslintrc`:

```json
{
  "extends": [
    "jsfile"
  ]
}
```

Then lint with `$ npm run lint`.
